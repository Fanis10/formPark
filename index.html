Каталог продукции

Предлагаем Вашему вниманию каталог продукции "Формы. Оборудование. Добавки к бетону".

В каталоге Вы найдете:
  • формы пластиковые для заливки тротуарной плитки производства STANDARTPARK и ALPHA (типы Брусчатка, Плиты, Газонные плиты, Садовые дорожки);
  • пластиковые формы для изготовления облицовочной плитки;
  • полиуретановые формы для производства искусственного камня и декоративных элементов;
  • эластичные формы из термопластичного полиуретана для производства тротуарной плитки и бетонных изделий;
  • пластиковые формы для производства декоративных изделий (бордюры, желоба, элементы инженерных коммуникаций);
  • формы для производства бетонных декоративных ограждений, заборов (односторонние и двухсторонние ограждения, столбы, камни-блоки);
  • формы для производства элементов садово-парковой архитектуры (малые архитектурные формы: балюстрады, шары, скамейки, вазы, урны, и другие);
  • дополнительные материалы для производства изделий из бетона (пигменты для бетона, добавки для бетона);
  • а также контакты наших офисов в Москве и регионах России.
